# Mobile-Sales-Dashboard

# Project Description
An interactive Power BI dashboard that analyzes mobile sales data to uncover key insights such as revenue trends, regional performance, and product-wise sales. The dashboard uses visualizations and filters to enable data-driven decision-making.

# Tech Stack
The dashboard was built using the following tools and technologies:
- Power BI Desktop - Main data visualization platform used for report creation.
- Power Query - Data Transformation and cleaning layer for reshaping and preparing the data.
- DAX (Data Analysis Expressions) - Used for calculated measures, dynamic visuals, and conditional logic.
- Data Modelling - Relationships established among tables to enable cross-filtering and aggregation.
- file Format - .pbix for development and .png for dashboard previews.

# 📊 Business Problem
The mobile sales business lacks a centralized system to monitor performance across regions, products, and time periods. Decision-makers face difficulty in:
- Identifying top-performing and underperforming products
- Tracking sales trends over time
- Comparing regional performance
- Making quick, data-driven decisions due to scattered or raw data
  
# 🎯 Goal of the Dashboard
It Provide a single interactive view of mobile sales data and Enable real-time analysis using filters and slicers. To Track key performance indicators (KPIs) such as total sales, revenue, and growth and Support data-driven decision-making for business strategy.

# 📊 Walkthrough of Key Visuals
1. KPI Cards
Displays Total Sales, Revenue, Profit (if included)
Gives a quick snapshot of overall business performance
2. Sales Trend (Line Chart)
Shows sales over time (monthly/quarterly)
Helps identify growth patterns, seasonality, and fluctuations
3. Sales by Region (Bar/Map Visual)
Compares performance across different regions
Highlights high-performing and low-performing areas
4. Product-wise Sales (Bar/Column Chart)
Displays sales distribution across different mobile models/brands
Identifies top-selling products
5. Category/Brand Distribution (Pie/Donut Chart)
Shows contribution of each category/brand to total sales
Useful for understanding market share
6. Filters / Slicers
Allows users to filter data by:
Region
Time period
Product/Brand
Enables interactive and customized analysis

# 📈 Business Impact
- Improves decision-making speed with visual insights
- Helps management focus on high-performing regions/products
- Identifies sales decline early, enabling corrective actions
- Enhances strategic planning and forecasting
- Reduces dependency on manual data analysis
